Knockback Knockoff is a 2D platformer game in which the player needs to navigate rooms using a diverse variety of guns to navigate through the map.

Brainstorming

This game started first as a a bug in an programming assigment. When first learning about rigidbody forces C# in Unity I was tasked with making a simple platformer. As the I was adding simple guns to the game I accidentally made the recoil too strong and the players were able to skip some sections through using the knockback of the guns to plaform around, and so the idea of Knockback Knockoff was born.

Process

Knockback knockoff is currently my longest lasting project as I've worked on it in and off due to college. I've been using this project as a learning experience to learn everything that I can about unity. Currently I've used this project to learn about the new input system, UI elements, programming my own physics and just programing in general!

I have devlogs about the project in the itch.io page with a playable demo so be sure to check it to get insights on the game and the programming journey of knockback knockoff
