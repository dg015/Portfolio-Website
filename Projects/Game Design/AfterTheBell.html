Role: Programmer, 3D artist

Email for contact: bruno.barbosa8@yahoo.com

Afte the bell is a first person boxing game that uses a fighting system inspired by Kingdom's Come Delivarance combat system.

Brainstorming

After the bell started as an idea after I impulse bought Kingdom Come Deliverace and ended up being too interested in the behind the scenes of the UI of the figthing system.

In college we were able to pick what to work on for our advanced programing and I chose to remake the UI system in Unity, after finishing it I had some extra days to spare so I added animations that would trigger based on which direction the player chose to attack. I enjoyed the idea of the project so much that I decided to pick it up and continue to work on it.
